Đáp án lựa chọn: Prompt 2

Lý do chọn Prompt 2:

Prompt 2 nêu mục tiêu rõ ràng: chuyển đổi từ vòng lặp for sang Java Stream API.
Prompt 2 nêu kết quả mong muốn cụ thể: code ngắn gọn, dễ bảo trì hơn.
Prompt 2 yêu cầu AI giải thích lý do tối ưu, giúp người học hiểu được sự thay đổi.
Đây là một prompt có ngữ cảnh, yêu cầu và ràng buộc rõ ràng, nên AI dễ tạo ra kết quả chính xác và phù hợp.

Vì sao Prompt 1 chưa tốt?

Prompt 1 chỉ nói: "Tối ưu hóa code Java này giúp tôi chạy nhanh hơn".
Yêu cầu quá mơ hồ, không chỉ rõ muốn tối ưu theo hướng nào (hiệu năng, độ ngắn gọn, khả năng bảo trì hay sử dụng công nghệ cụ thể).
AI có thể đưa ra nhiều cách khác nhau và không chắc sẽ chuyển sang Java Stream API như mong muốn.
